# Sesión 8 — Taller de aplicación final

> Todo junto, sobre tu propia investigación, de principio a fin.

La sesión de cierre es un **taller integrador**. No hay contenido nuevo: vas a aplicar todo lo aprendido a un problema real tuyo, recorriendo el flujo completo —pedir, ejecutar, analizar, visualizar y validar— sobre tus propias fuentes.

## Objetivos

Al terminar esta sesión vas a poder:

- **Definir** un problema propio abordable con estas herramientas.
- **Aplicar** el flujo completo del curso a tus datos.
- **Validar y documentar** el resultado.
- **Presentar** lo que hiciste con criterio.

## Contenidos

- Definición de un problema de investigación propio.
- Aplicación integrada: del corpus al gráfico final.
- Validación y documentación del proyecto.
- Puesta en común de resultados.

## Para practicar

- Llegá con una pregunta y un conjunto de datos tuyos.
- Al cierre, dejá tu proyecto documentado para poder retomarlo más adelante.
