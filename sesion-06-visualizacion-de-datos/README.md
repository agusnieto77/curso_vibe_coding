# Sesión 6 — Visualización de datos con IA

> Mostrar lo que encontraste. Un buen gráfico dice más que una tabla enorme.

Los números solos no comunican. En esta sesión vamos a **visualizar** los resultados del análisis: convertir conteos y patrones en gráficos que se entiendan de un vistazo y sirvan para tus publicaciones o presentaciones.

## Objetivos

Al terminar esta sesión vas a poder:

- Elegir el **tipo de gráfico** adecuado para cada dato.
- Generar **gráficos de barras** para comparar frecuencias.
- Generar **series temporales** para ver evolución en el tiempo.
- Generar **nubes de palabras** para una mirada rápida del corpus.

## Contenidos

- Qué gráfico usar y cuándo.
- Gráficos de barras.
- Series temporales.
- Nubes de palabras.
- Ajustar y exportar para publicar.

## Para practicar

- Tomá un resultado de la sesión anterior y pedí dos gráficos distintos del mismo dato.
- Decidí cuál comunica mejor y por qué.
