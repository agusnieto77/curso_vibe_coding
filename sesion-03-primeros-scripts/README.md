# Sesión 3 — Primeros scripts sin saber programar

> Tu primer script, sin pánico. Lo importante no es escribirlo: es entenderlo.

Ya sabemos pedir; ahora vamos a **ejecutar**. En esta sesión vas a correr tu primer script generado por IA, entender su estructura básica y, sobre todo, animarte a **modificarlo** sin miedo a romper nada.

## Objetivos

Al terminar esta sesión vas a poder:

- Leer la **estructura básica** de un script y saber qué hace cada parte.
- **Ejecutar** un script en tu computadora.
- **Modificar** el código generado (cambiar valores, ajustar comportamientos).
- Pedirle a la IA que explique o corrija un script que no funciona.

## Contenidos

- Qué es un script y cómo se ejecuta.
- Las partes básicas de un script (entrada, proceso, salida).
- Cómo cambiar valores sin romper el código.
- Qué hacer cuando algo da error.

## Para practicar

- Pedí un script chico (por ejemplo, contar palabras de un archivo de texto).
- Ejecutalo, cambiá un valor a propósito y observá qué pasa.
