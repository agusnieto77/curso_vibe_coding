# Vibe Coding para Humanistas

> Programar con IA sin saber programar.

Curso para investigadoras e investigadores de **Humanidades y Ciencias Sociales** que quieren usar herramientas de inteligencia artificial para escribir código en **lenguaje natural** —sin formación previa en programación— y aplicarlo a su propia investigación: limpiar corpus, automatizar tareas, analizar texto y visualizar datos.

---

## ¿Qué vas a aprender?

Al terminar el curso vas a poder:

- Entender qué es el **vibe coding** y hasta dónde llega en la investigación académica.
- Formular instrucciones claras en lenguaje natural para obtener **código útil**.
- Generar **scripts básicos** para analizar corpus textuales.
- **Automatizar** tareas repetitivas de investigación.
- Producir **visualizaciones de datos**.
- Aplicar criterios de **validación y documentación** del código que genera la IA.

## ¿Para quién es?

Investigadoras e investigadores en Humanidades y Ciencias Sociales que trabajan con texto y datos y quieren incorporar la IA a su flujo de trabajo. **No hace falta saber programar.**

## Antes de empezar

Para seguir el curso vas a necesitar:

- [ ] Ordenador personal con conexión estable a internet.
- [ ] Navegador web actualizado.
- [ ] Cuenta activa en una herramienta de IA (ChatGPT, Claude o equivalente; la versión gratuita alcanza).
- [ ] Disposición para instalar software básico.
- [ ] *Recomendable:* una segunda pantalla o ventanas divididas.
